
## 코딩테스트 관련 확인 사항
### 문자열

- 문자열 수변환
    - 정수 변환 : Integer.parseInt(s)
- 다른 자료형을 문자열로 변환
    - String.valueOf()
    - +””
- 엔덱싱
    - charAt(i)
- 주요 메소드
    - length()
    - substring()
        - substring(1,4) 라면 1~3 까지
    - toUpperCase(), toLowerCas()
    - trim : 양쪽에 있는 공백 및 기타 공백 문자 제거
    - replace()
    - indexOf(), lastIndexOf() : 문자열 찾기( 못찾으면 -1 리턴)
- 특수문자
    - `\n`  (새 줄), `\t`  (탭),`\\`  (백슬래시),`\'`  (작은따옴표),`\"`  (큰따옴표)
- 여러값 동시에 선언
    - `int a=2, b=3;`
- 변수값 동시 복사
    - `a = b = c`
        - b =c  넣어준 후 a =b
- System.out.printf
    - 정수 `%d`
    - 실수 `%f`
        - 2번째 자리 수 설정 `%.2f`
    - 문자열 `%s`
    - 줄 바꿈 `%n`
    
- continue
    - 다음 반복문으로
- break
    - 반복문 종료
    - 내부 반복문에서 사용하면 내부 루프만 종료
    - 외부 루프에서는 전체 종료



### Scanner 주의점

- `nextInt()`
    - 뛰어쓰기 하든 행을 바꿔쓰든 상관없이 숫자를 가져옴 nextLine 굳이 안써도됨
    - 숫자만 많으면 → nextInt()만 써
    - 문자열 섞이면 → 그냥 전부 nextLine()으로 통일하고 파싱해도 됨
    - nextInt() 다음에 nextLine()이면 남은 엔터 제거 위해 nextLine() 한 번 써주기
- `next()`
    - 문자열 한 개 입력
    - next().charAt(0) :  첫번째 문자 확인
